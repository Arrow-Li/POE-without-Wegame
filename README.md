# POE-without-Wegame
1. Compile & Install
  
    1. run `mingw32-make`
    2. copy `poe.exe` to `xxx\Path of Exile\WeGameLauncher`

2. Create Shortcut

    * Target File:`"xxx\Path of Exile\WeGameLauncher\poe.exe" "xxx\\Path of Exile\\logs\\Client.txt"`
                  
    * Start In:`"xxx\Path of Exile\WeGameLauncher"`
